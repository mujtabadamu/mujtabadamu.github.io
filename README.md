# mujtabadamu.github.io

My personal portfolio site, built with plain HTML/CSS/JS and hosted on GitHub Pages.

Live at: https://mujtabadamu.github.io

## Editing

- `index.html` — content and structure. Sections marked `<!-- TODO -->` are placeholders to replace with real info.
- `style.css` — styling, including light/dark theme variables at the top.
- `script.js` — theme toggle and footer year.

## Deploying

This is a GitHub **user site**, so it deploys automatically from the `main` branch of this repo — no build step or GitHub Actions needed. Push to `main` and the site updates at https://mujtabadamu.github.io within a minute or two.
